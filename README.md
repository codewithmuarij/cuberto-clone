🚀 Cuberto Landing Page Clone
📌 Overview

This is a Cuberto-inspired animated landing page built to practice and demonstrate modern frontend development skills, focusing heavily on smooth motion design, scroll-based interactions, and UI animation principles.

The goal of this project was not just to replicate a design, but to understand how high-end agency websites create immersive user experiences using animation, timing, and scroll behavior.

⚙️ Tech Stack
HTML5 – semantic structure
CSS3 – layout, styling, responsiveness
JavaScript (ES6+) – DOM interaction and animation control
GSAP (GreenSock Animation Platform) – advanced animations and timelines
Lenis – smooth scrolling experience

✨ Key Features
🎬 Smooth Scroll Experience

Implemented Lenis smooth scrolling to create a fluid, modern navigation feel that enhances the overall user experience and makes scroll-based animations more natural.

🌀 Advanced GSAP Animations
Timeline-based animations for structured motion flow
Scroll-triggered effects for dynamic storytelling
Smooth entrance transitions for sections and elements
Micro-interactions to enhance UI responsiveness

🎨 Modern UI/UX Feel
Minimal, clean layout inspired by Cuberto’s design philosophy
Focus on typography and spacing for visual hierarchy
Smooth motion transitions for premium feel
Attention to interaction details (hover, scroll, reveal effects)

This project helped me understand:

How to structure GSAP timelines for real-world UI
How scroll behavior impacts user experience
How to combine Lenis + GSAP for smooth motion systems
How professional landing pages use subtle animations instead of heavy effects
Performance considerations when animating DOM elements

🚧 Challenges Faced
Synchronizing GSAP animations with smooth scroll behavior
Maintaining performance while using multiple animated elements
Structuring animations in a scalable and readable way
Ensuring responsiveness without breaking animation flow
